# Design Context

This project is a College Attendance Management System. Key design decisions live in:

- **PRODUCT.md** — strategic: register (product), users, brand personality ("clean, fast, no-nonsense"), anti-references, design principles, accessibility
- **DESIGN.md** — visual: creative north star ("The Quiet Register"), Restrained color strategy (navy-slate accent), single sans typography, flat-by-default elevation

Before building any UI surface, read both files. The register is **product** — design serves the task, not the brand.
